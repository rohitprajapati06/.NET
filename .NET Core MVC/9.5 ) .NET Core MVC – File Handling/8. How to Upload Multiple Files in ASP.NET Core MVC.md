# How to Upload Multiple Files in ASP.NET Core MVC

Multiple file upload allows users to upload more than one file in a single request.

Common use cases:

* Product Images
* User Documents
* Medical Reports
* Gallery Management
* Employee Certificates
* Travel Documents

---

# How Multiple File Upload Works

```text
User Selects Files
        ↓
Browser Sends Files
        ↓
IFormFile Collection
        ↓
Controller Receives Files
        ↓
Save Files
        ↓
Store File Paths in Database
```

---

# Controller Action

```csharp
public class FileController : Controller
{
    private readonly IWebHostEnvironment _environment;

    public FileController(
        IWebHostEnvironment environment)
    {
        _environment = environment;
    }

    [HttpPost]
    public async Task<IActionResult> UploadFiles(
        FileUploadViewModel model)
    {
        if (model.Files == null ||
            !model.Files.Any())
        {
            ModelState.AddModelError(
                "",
                "Please select files.");

            return View();
        }

        string uploadFolder =
            Path.Combine(
                _environment.WebRootPath,
                "uploads");

        Directory.CreateDirectory(
            uploadFolder);

        foreach (var file in model.Files)
        {
            if (file.Length > 0)
            {
                string uniqueFileName =
                    Guid.NewGuid().ToString()
                    + Path.GetExtension(
                        file.FileName);

                string filePath =
                    Path.Combine(
                        uploadFolder,
                        uniqueFileName);

                using var stream =
                    new FileStream(
                        filePath,
                        FileMode.Create);

                await file.CopyToAsync(
                    stream);
            }
        }

        ViewBag.Message =
            "Files uploaded successfully.";

        return View();
    }
}
```

# Understanding the Code

## Loop Through Files

```csharp
foreach (var file in model.Files)
{
    ...
}
```

Processes each uploaded file individually.

---

## Generate Unique File Name

```csharp
Guid.NewGuid()
```

Example:

```text
invoice.pdf
```

becomes

```text
4f2f67d2-8b5e.pdf
```

This prevents overwriting.

---

## Save File

```csharp
await file.CopyToAsync(stream);
```

Writes the file to disk.

# Upload Using IFormFileCollection

Instead of a ViewModel:

```csharp
[HttpPost]
public async Task<IActionResult> UploadFiles(
    IFormFileCollection files)
{
    foreach (var file in files)
    {
        // Save file
    }

    return View();
}
```

# Validate File Extensions

```csharp
string[] allowedExtensions =
{
    ".jpg",
    ".jpeg",
    ".png",
    ".pdf"
};

foreach (var file in model.Files)
{
    string extension =
        Path.GetExtension(
            file.FileName)
            .ToLower();

    if (!allowedExtensions.Contains(
            extension))
    {
        ModelState.AddModelError(
            "",
            $"{file.FileName} is invalid.");
    }
}
```

---

# Validate File Size

Example: Maximum 5 MB per file.

```csharp
const long maxSize =
    5 * 1024 * 1024;

foreach (var file in model.Files)
{
    if (file.Length > maxSize)
    {
        ModelState.AddModelError(
            "",
            $"{file.FileName} exceeds 5 MB.");
    }
}
```

---

# Limit Number of Files

Example: Maximum 10 files.

```csharp
if (model.Files.Count > 10)
{
    ModelState.AddModelError(
        "",
        "Maximum 10 files allowed.");
}
```

# Store File Details in Database

## Entity

```csharp
public class UploadedFile
{
    public int Id { get; set; }

    public string OriginalFileName { get; set; }

    public string StoredFileName { get; set; }

    public string FilePath { get; set; }

    public DateTime UploadedOn { get; set; }
}
```

---

## Save Metadata

```csharp
var uploadedFile =
    new UploadedFile
    {
        OriginalFileName =
            file.FileName,

        StoredFileName =
            uniqueFileName,

        FilePath =
            filePath,

        UploadedOn =
            DateTime.UtcNow
    };

_context.UploadedFiles
    .Add(uploadedFile);
```

After all files:

```csharp
await _context.SaveChangesAsync();
```

# Upload Files to Azure Blob Storage

Since you've worked with Azure Blob Storage, a production-ready approach is:

```text
User Upload
      ↓
ASP.NET Core
      ↓
Azure Blob Storage
      ↓
Store Blob URL in Database
```

Example:

```csharp
await blobClient.UploadAsync(
    file.OpenReadStream(),
    true);
```

Save:

```csharp
blobClient.Uri.ToString()
```

to the database.

# Interview Questions

### How do you upload multiple files in ASP.NET Core MVC?

Use:

```csharp
List<IFormFile>
```

or

```csharp
IFormFileCollection
```

along with the HTML `multiple` attribute.

---

### Why use `multipart/form-data`?

Because it allows files and form data to be sent together in the HTTP request.

---

### How do you prevent file overwriting?

Generate unique names using:

```csharp
Guid.NewGuid()
```

instead of saving files with their original names.

---

### What is the recommended production storage approach?

Store files in:

* Azure Blob Storage
* AWS S3
* Google Cloud Storage

and save only the file URL/path in the database.

---

### How would you implement multiple product images in an e-commerce application?

Create a separate `ProductImages` table with a foreign key to `Products`, then upload and store multiple image records for a single product.
