# Token Replacement in ASP.NET Core Web API

Token replacement is a feature of **Attribute Routing** that allows ASP.NET Core to automatically substitute placeholders in route templates with values from controllers, actions, or areas.

It helps avoid hardcoding route names and makes routes easier to maintain.

---

## Available Tokens

| Token          | Replaced With                                   |
| -------------- | ----------------------------------------------- |
| `[controller]` | Controller name without the `Controller` suffix |
| `[action]`     | Action method name                              |
| `[area]`       | Area name                                       |

---

# 1. `[controller]` Token

Consider this controller:

```csharp
[ApiController]
[Route("api/[controller]")]
public class HospitalsController : ControllerBase
{
    [HttpGet]
    public IActionResult Get()
    {
        return Ok();
    }
}
```

ASP.NET Core replaces:

```text
[controller]
```

with:

```text
Hospitals
```

Resulting route:

```http
GET /api/hospitals
```

---

### Another Example

```csharp
[ApiController]
[Route("api/[controller]")]
public class DoctorsController : ControllerBase
{
    [HttpGet]
    public IActionResult Get()
    {
        return Ok();
    }
}
```

Generated route:

```http
GET /api/doctors
```

# 2. `[action]` Token

```csharp
[ApiController]
[Route("api/[controller]/[action]")]
public class HospitalsController : ControllerBase
{
    [HttpGet]
    public IActionResult GetAll()
    {
        return Ok();
    }

    [HttpGet]
    public IActionResult GetPending()
    {
        return Ok();
    }
}
```

Generated routes:

```http
GET /api/hospitals/GetAll

GET /api/hospitals/GetPending
```

---

## Renaming an Action

Suppose you rename:

```csharp
GetAll()
```

to:

```csharp
GetHospitals()
```

The route automatically changes to:

```http
GET /api/hospitals/GetHospitals
```

No route template updates are required.

---

# 3. `[area]` Token

Areas are mostly used in MVC applications, but they also work with APIs.

```csharp
[Area("Admin")]
[ApiController]
[Route("api/[area]/[controller]")]
public class HospitalsController : ControllerBase
{
    [HttpGet]
    public IActionResult Get()
    {
        return Ok();
    }
}
```

Generated route:

```http
GET /api/admin/hospitals
```

---

# 4. Token Replacement in Route Names

Tokens can also be used for route names.

```csharp
[ApiController]
[Route("api/[controller]")]
public class HospitalsController : ControllerBase
{
    [HttpGet("{id}", Name = "[controller]_[action]")]
    public IActionResult Get(Guid id)
    {
        return Ok();
    }
}
```

ASP.NET Core generates the route name:

```text
Hospitals_Get
```

# Example from Your SmartHealthcare Project

Your controller is similar to:

```csharp
[ApiController]
[Route("api/[controller]")]
public class HospitalsController : ControllerBase
{
    private readonly IMediator mediator;

    public HospitalsController(IMediator mediator)
    {
        this.mediator = mediator;
    }

    [HttpGet]
    public async Task<IActionResult> Get()
    {
        return Ok(await mediator.Send(
            new GetHospitalsQuery()));
    }

    [HttpGet("{id:guid}")]
    public async Task<IActionResult> Get(Guid id)
    {
        return Ok(await mediator.Send(
            new GetHospitalByIdQuery(id)));
    }
}
```

Generated routes:

| Action         | URL                     |
| -------------- | ----------------------- |
| `Get()`        | `/api/hospitals`        |
| `Get(Guid id)` | `/api/hospitals/{guid}` |

# Best Practices

### Prefer `[controller]`

```csharp
[Route("api/[controller]")]
```

Instead of:

```csharp
[Route("api/hospitals")]
```

because renaming `HospitalsController` to `MedicalCentersController` automatically changes the route to:

```http
/api/medicalcenters
```

---

### Avoid `[action]` in REST APIs

REST APIs typically model **resources**, not actions.

Prefer:

```http
GET    /api/hospitals
GET    /api/hospitals/{id}
POST   /api/hospitals
PUT    /api/hospitals/{id}
DELETE /api/hospitals/{id}
```

Avoid:

```http
/api/hospitals/GetAll
/api/hospitals/Create
/api/hospitals/Delete
```

---

# Interview Answer

> Token replacement in ASP.NET Core routing is the process of replacing placeholders such as `[controller]`, `[action]`, and `[area]` with their corresponding runtime values when building route templates. It reduces hardcoded route strings and improves maintainability.
